1️⃣ 화재안전기준 공부하기 (NFTC 학습카드)
   ▸ NFPC/NFTC 101~501 전 기준을 플래시카드로 학습
   ▸ 6단계 암기 평가 (모른다·중요·기억·메모·무시·안다)
   ▸ 홈 화면에 추가하면 앱처럼 사용 (PWA)

2️⃣ 녹음 기록기 (Voice Memo)
   ▸ 점검 현장에서 음성 녹음 + 표 형식 메모 + 사진 첨부
   ▸ 카카오톡으로 결과 즉시 공유
   ▸ 별도 설치 없이 브라우저에서 바로 사용
   ▸ 직접 사용: https://hankal0501-ux.github.io/voice-memo/

3️⃣ 반복 학습 플레이어 (Repeat Player)
   ▸ 영상을 무한 반복 재생 (없음/2x/3x/5x/∞)
   ▸ 로컬 영상 파일 + YouTube URL 모두 지원
   ▸ 외국어·소방 안전교육 영상 학습에 최적
   ▸ 직접 사용: https://hankal0501-ux.github.io/repeat-player/
